Sistem Laporan Keuangan Warga Komplek PGRI

Solusi digital untuk pencatatan dan pengelolaan keuangan warga komplek dengan sistem keamanan admin yang terintegrasi dengan Google Apps Script.

Aplikasi Utama Pelajari Lebih Lanjut

Fitur Unggulan

Keamanan Admin

Sistem login admin yang membatasi akses tambah/hapus transaksi hanya untuk pengguna terautentikasi.

Integrasi Cloud

Data tersimpan aman di Google Spreadsheet melalui Google Apps Script, dapat diakses dari mana saja.

Laporan Real-time

Ringkasan keuangan otomatis dengan perhitungan saldo, pemasukan, dan pengeluaran secara real-time.

Cara Kerja

1
Setup Google Apps Script

Buat Google Spreadsheet dan deploy Google Apps Script sebagai backend untuk menyimpan data transaksi.

2
Konfigurasi Aplikasi

Masukkan URL Google Apps Script ke aplikasi dan simpan konfigurasi untuk koneksi ke database.

3
Login sebagai Admin

Gunakan kredensial admin untuk mendapatkan akses penuh dalam mengelola transaksi keuangan.

4
Kelola Transaksi

Tambahkan pemasukan/pengeluaran, hapus transaksi, dan export data laporan keuangan.

Instalasi

1. Clone Repository

git clone https://github.com/username/laporan-keuangan-pgri.git
cd laporan-keuangan-pgri

2. Setup Google Apps Script

Buat Google Spreadsheet baru dengan nama "Laporan Keuangan PGRI" dan sheet "Transaksi". Tambahkan header: ID, Tanggal, Keterangan, Jenis, Jumlah.

3. Deploy Apps Script

Gunakan kode Google Apps Script yang telah disediakan dan deploy sebagai web app dengan akses "Anyone".

4. Konfigurasi URL

Masukkan URL web app yang sudah di-deploy ke aplikasi dan simpan konfigurasi.
